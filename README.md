# CAP_rCCA_prediction
code for manuscript fear learning induced CAP...

This repository contains analysis scripts used in the study.

## Modules

- connectivity: functional connectivity computation
- cap: co-activation pattern analysis
- hmm: hidden Markov model analysis
- entropy: trajectory entropy analysis
- prediction: rCCA prediction model

## Dependencies

Python = Python 3.11.6

Packages:
- numpy
- scipy
- scikit-learn
- nilearn
- cca-zoo
- matplotlib
- seaborn
